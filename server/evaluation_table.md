| Evaluation Metric                   | Formula                                                                                             | Evaluation Goal                                             |
|-------------------------------------|-----------------------------------------------------------------------------------------------------|-------------------------------------------------------------|
| **Sparsity (Gini Index)**           | \( G = \frac{\sum_{i=1}^{n} \sum_{j=1}^{n} | x_i - x_j |}{2n \sum_{i=1}^{n} x_i} \)                | Measures the concentration of important features; higher values indicate sparser maps. |
| **Relative Input Stability (RIS)**  | \( dRIS = RISM[VG(x)] - RISN[VG(x)] \)                                                              | Tests the stability of the saliency map with respect to input perturbations. |
| **Relative Output Stability (ROS)** | \( dROS = ROSM[VG(x)] - ROSN[VG(x)] \)                                                              | Tests the stability of the saliency map with respect to output perturbations. |
| **Relative Representation Stability (RRS)** | \( dRRS = RRSM[VG(x)] - RRSN[VG(x)] \)                                                     | Tests the stability of feature representations in the saliency map under input perturbations. |
| **Structural Similarity Index (SSIM)** | \( \text{SSIM}(x, y) = \frac{(2 \mu_x \mu_y + c_1)(2 \sigma_{xy} + c_2)}{(\mu_x^2 + \mu_y^2 + c_1)(\sigma_x^2 + \sigma_y^2 + c_2)} \) | Evaluates the similarity between two saliency maps, assessing robustness under slight input noise. |
| **Faithfulness Estimate**           | \( \text{Faithfulness Estimate} = \text{confidence}(x) - \text{confidence}(x_{\text{perturbed}}) \) | Checks if the saliency map accurately reflects critical features that contribute to the modelâ€™s prediction. |
| **Softmax Information Curve (SIC)** | -                                                                                                   | Measures the model's softmax probability change as important features are removed. |
| **Accuracy Information Curve (AIC)**| -                                                                                                   | Measures the change in model accuracy as important features are removed. |
| **Sensitivity**                     | \( \Delta = ||V G_F(x') - V G_F(x)||_1 \leq (F(x') - F(x)) \cdot w \)                               | Tests saliency map's sensitivity to input noise, ensuring stability under minor perturbations. |
| **Removal of Most Relevant Features (MoRF)** | \( \text{Accuracy drop} = \text{accuracy}(x) - \text{accuracy}(x_{\text{MoRF}}) \)          | Tests the impact of removing key features from the saliency map on model accuracy, indicating feature discriminativeness. |
